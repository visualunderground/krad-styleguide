Breadcrumbs are used to show a user's hierarchical location within the site and not, necessarily, the path taken.

Breadcrumbs are used in conjunction with the content header to inform users of what they're doing - and where they're doing it.

  1. breadcrumbs are not used on the Home page
  2. the first item must always be a link to Home
  3. the last item must never be a link
  4. when in an AE/VTS the breadcrumbs should reflect the entities' name
  5. task 'steps' must not be shown in the breadcrumb trail
