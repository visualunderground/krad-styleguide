Use white space to create a visual hierarchy on the page. Design for small screens first using a single column layout.

Spacing

  • spacing between elements is usually 5px, 10px, 15px or multiples of 15px
  • gutters are 15px for smaller screens and 30px for larger screens

Grid layout

Grid classes enable easy cross browser grids. The grids use absolute widths in older versions of IE or percentage based widths in modern browsers.

Usage

<main role="main">
    <div class="grid-row">
        <div class="column-two-thirds">
            Main content
        </div>
        <div class="column-third">
            Sidebar
        </div>
    </div>

    <div class="grid-row">
        <div class="column-third">
            Column one
        </div>
        <div class="column-third">
            Column two
        </div>
        <div class="column-third">
            Column three
        </div>
    </div>
</main>
