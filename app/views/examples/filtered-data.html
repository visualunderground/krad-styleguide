This page demonstrates how to display filtered data in the MOT system.

  1. A summary of the number of results, applied filters or search terms should be including above the table
  2. Column headers should use the th tag
  3. Sortable columns should be indicated using a hyperlink
  4. The default sort should be sensible and indicated with a sort arrow
  5. If required, download options should appear in sentence format below the results table
{{> mot_a_filtered-data }}

CSS class summary

A summary of the CSS classes used is as follows:

  1. Apply sort-desc and sort-asc to the sort hyperlink to indicate appropriate sort direction
  2. Numerical data (and associated column header) should be formatted using the standard numeric class
  3. Other data type values should have the tabular class applied which negates right align styles

Results table with filter boxes

  1. When page includes filters, a 33/66 column split should be used
  2. For accessibility and content order purposes, the results table should be first in content
  3. For currently selected filters, remove the hyperlink and show as bold text
